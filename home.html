The Crowbar Project

The Crowbar Project is an effort to build a complete, easy to use operational platform for everyone. It allows for groups of physical nodes to be transformed from bare-metal into a production cluster within hours. Specific applications include (but are not limited to) Hadoop and OpenStack.
