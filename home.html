The Crowbar Project

The Crowbar Project is an effort to build a complete, easy to use operational platform for everyone. It allows for groups of physical nodes to be transformed from bare-metal into a production cluster within hours. Specific applications include (but are not limited to) Ceph, Hadoop and OpenStack.

Get involved with our community!

  • Sign up for the group / mail list: Group Groups OpenCrowbar
  • Hang out with us on IRC: Freenode channel #crowbar
  • File Defects or Create Features: Github Issues

The Crowbar Project has a single community with two versions under active development.

Crowbar v1 OpenCrowbar (aka v2)
The original Chef integrated provisioning tool. Since 2014, primarily maintained by SUSE for OpenStack deployments. The re-architected replacement for Crowbar v1 supports a much broader set of operating systems, applications, configuration managers and hardware (physical, virtual and containers). OpenCrowbar is community maintained.
