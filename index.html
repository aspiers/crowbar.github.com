--- layout: default title: Modular provisioning platform ---

An openly licensed framework to build complete, easy to use operational deployments. It allows for groups of physical nodes to be transformed from bare-metal into a ready state production cluster within hours. Originally founded in 2011 by Dell, since 2014 primarily maintained by SUSE for OpenStack deployments.

Features include CEPH, High Availability, SSL Support and unattended installation.
