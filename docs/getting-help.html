Crowbar IRC Channel

The channel is #crowbaron FreeNode .

  • You can connect via the irc.freenode.net server , or via webchat
  • Channel logs are also available.

Crowbar Mailing List

Crowbar mailing list: http://lists.us.dell.com/mailman/listinfo/crowbar (ignore any SSL certificate warnings).

All communication to members who are subscribed to the list can go to crowbar@lists.us.dell.com. However, the mailing server is configured so that the “From” header will appear as crowbar@dell.com (instead of crowbar@lists.us.dell.com), and so all sub-sequent replies to this address will fail since it does not exist in the Dell corporate Exchange servers.

We are working to address this issue and it should be resolved by launch.
